tipos de animaciones

desplazamiento horizontal

@keyframes slide-right {
from {
transform: translateX(-100%);
}
to {
transform: translateX(0);
}
}
.element {
animation: slide-right 2s ease-in-out;
}

desplazamiento vertical

@keyframes slide-down {
from {
transform: translateY(-100%);
to {
transform: translateY(0);
}
}
.element {
animation: slide-down 2s ease-in-out;
}

rotacion

@keyframes rotate {
from {
transform: rotate(0deg);
}
to {
transform: rotate(360deg);
}
}
.element {
animation: rotate 1s linear infinite;
}

cambiar color

}

}
}

}

esta pro[iedad declara la posicon de comienzo de una imagen de fondo los valores se pueden especificar en procentaje, pixeles o usando una combinacion de las palabra center,left,right,top y bottom

BORDES

por defecto los navegadores no muestran ningun borde, pero podemos usar las siguientes porpiedades para definirlo

border-width

esta propiedad define el ancho del borde acepta cuatro valores por defecto border-top-width , border-bootom-width , border-left-width y border-right-width

border-style

esta propiedes define el estili del borde acepta cuatro valores todos lo valores que voy diciendo van en el siguiente orden que es el superior,derecho,inferio e izquierdo en ese orden los valores son los siguientes none,hidden,dotted,dashed,solid,double,groove,ridge,inset y outsetel valor por defecto es none lo que significa que no se mostrara a menos que le mostremos algun valor se puede asignar los valores de forma independiente border-top-style , border-botton-style , border-left-style y border-right-style-

border-color

esta propiedad define el color del borde y va acompanado con los anteriores valores por ejemplo border-top-color
